Welcome to the site! This is an introduction to help you get started.

## What is this?

This is a documentation and blog site built with Go. It features:

- Markdown content with custom directives
- Syntax highlighting for code blocks
- Table of contents navigation
- Emoji reactions

## Code Examples

Here's a simple Go program:

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

And some JavaScript:

```javascript
const greet = (name) => {
  console.log(`Hello, ${name}!`);
};

greet("World");
```
